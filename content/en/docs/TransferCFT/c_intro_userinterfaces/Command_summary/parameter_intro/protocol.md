### protocol

####  

#### CFTNET TYPE = TCP

PROTOCOL = { '  '
| SOCKS4| SOCKS5 }

Defines the remote TCP network resource proxy used to access the second
network:

- SOCKS4
- SOCKS5
- ' ' a blank indicates no proxy protocol is defined
