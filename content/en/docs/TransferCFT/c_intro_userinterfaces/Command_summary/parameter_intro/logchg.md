#### WLOG

logchg = { NONE | STATIC | DYNAMIC | ALL
}

Log configuration changes:

-   none - no logging
    is performed
-   static - only the
    static configuration is logged
-   dynamic - only
    the dynamic configuration is logged
-   all - log all changes
