}1.  In the Composer Integration-Services workbench / **Integration Manager** tab, expand the directory tree in the left pane and right-click the Integration-Task.
2.  From the context menu, select **Advanced actions > Send to Server…**  
    Composer sends the Integration-Task and the associated objects (Integration-Process, Business-Documents, Map-Brokers, ...) to the Integrator Server that is specified in the Integration-Task configuration.
