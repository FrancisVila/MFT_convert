}Gateway product is delivered with the CA certificate needed to connect to Passport product (Passport SSL CA certificate). This certificate is delivered in <span class="code">extras\\PassPort\\</span> directory of the installation and corresponding configuration parameters for accessing the CA certificate are defined to this path by default.

Gateway does not deliver sample certificates.
