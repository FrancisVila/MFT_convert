}The SecureTransport on Azure Installation Guide provides a detailed overview and instructions to set up SecureTransport in the Microsoft Azure portal.

Microsoft Azure lets you provision a logically isolated section of the Azure where you can launch Azure resources in a virtual network that you define. You have complete control over your virtual networking environment, including selection of your own IP address range, creation of subnets, and configuration of route tables and network gateways.
