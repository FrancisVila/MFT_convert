}Use an exact IPv4 or IPv6 address to specify a single server. In IPv6 addresses, two colons (`::`) can represent one sequence of zero bits.

Examples of valid values include the following:

-   `172.23.34.45`
-   `127.0.0.1`
-   `FC00:1234:56:0:0:0:AB:EF`
-   `FC00:1234:56::AB:EF`
-   `::1`
