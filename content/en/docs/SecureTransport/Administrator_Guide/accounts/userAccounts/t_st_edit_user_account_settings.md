> If you are using a FireFox browser, disable the auto complete function before editing user account settings.

Use the following procedure to edit user account settings.

1.  Select **Accounts > User Accounts**.  
    The *User Accounts* page is displayed.

2.  Click the name of the account whose settings you want to edit.  
    The *User Account Settings* page is displayed with details for the selected account.

3.  Click **Edit Account Settings**.  
    The *Edit Account Settings* page is displayed.  

    > **Note:**
    >
    > The Address Book Settings are only displayed if the Address Book feature is enabled (the value of the AddressBook.Enabled configuration option is set to true). For Address Book account level configuration instructions, refer to Address Book account level configuration.

      
    <img src="/Images/SecureTransport/edit-account-settings.png" class="maxWidth" alt="Edit Account Settings" />

4.  Edit the account settings. For information about the fields on the *Edit Accounts Settings* page, see [Create a user account](../t_st_create_user_account#AccountsMenu_2253641766_1058767).

5.  To add attributes:
    1.  Click the **Add Attribute** button.
    2.  Complete the **Attribute** and **Value** fields.
    3.  Click the attribute Save (![](/Images/SecureTransport/SaveIcon.png)) icon.

6.  To delete an attribute or attributes:
    1.  Select the attribute or attributes to delete.
    2.  Click the attribute **Delete** button.

7.  To edit an attribute:
    1.  Select the attribute to edit.
    2.  Click the attribute Edit (![](/Images/SecureTransport/EditIcon.png)) icon.
    3.  Make the desired attribute changes.
    4.  Click the attribute Save (![](/Images/SecureTransport/SaveIcon.png)) icon.
